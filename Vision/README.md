### Google Images 크롤링
- dependency 설치
~~~
pip install -r requirements.txt
~~~
- 프로그램 사용
~~~
python crawler.py
python3 crawler.py // if using python3 keyword
~~~
  - 현재 디렉토리의 downloads 폴더에 성별/동물상 별 이미지 50개씩 다운로드
  - Used Package: https://github.com/hardikvasa/google-images-download
